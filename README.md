# はたらくくるま — あそびのえほん 🚗🖍️

子供向けのブラウザゲーム。たくさんの小さな車が地面を左から右へ走り、
プレイヤーは指（マウス/タッチ）で障害物を描きます。車はそれを乗り越えながら進み、
工場車両（削り車・トンネル車）は障害物をなめらかにしたり横穴を掘ったりします。
やわらかいクレヨン絵本のような世界観（案A）。

**完全な静的サイト**（サーバ処理なし・ビルド不要・依存ライブラリなし）。
HTML5 Canvas + バニラ JS（ES モジュール）。詳しい仕様は [`CLAUDE.md`](./CLAUDE.md)。

## あそびかた

- 画面を指やマウスで**なぞると障害物ブロックが描ける**（空中に描くと落ちて積み重なる）
- 車はブロックを乗り越えて右へ。急な壁は越えられず止まる
- **削り車**（研削ディスク）は壁を削ってなめらかに、**トンネル車**（ドリル）は横穴を掘ってバイパスを作る
- 右上の **「ぜんぶけす」** ボタンで障害物だけを全部消す（車は残る）

## ローカルでの確認

ES モジュールは `file://` 直開きでは動きません。簡易サーバで開いてください。

```bash
python3 -m http.server
# → ブラウザで http://localhost:8000/
```

スマホ/タブレット実機でタッチ操作と動作の軽さも確認してください。

## GitHub Pages で公開

1. このファイル一式を**リポジトリのルート**に置いて `main` に push
2. ルートに空の `.nojekyll` があること（Jekyll 処理の無効化・同梱済み）
3. GitHub → **Settings → Pages**
4. **Source = “Deploy from a branch”**、**Branch = `main` / `(root)`** を選んで Save
5. 数十秒後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開

すべて**相対パス**で書いているため、サブパス配信でも崩れません。

## ファイル構成

```
index.html              … Canvas と起動
.nojekyll               … GitHub Pages の Jekyll 無効化
src/config.js           … 調整用の定数・カラートークン
src/grid.js             … セルグリッド/クラスタ/重力/削り/掘削
src/cars.js             … 車の種類・スポーン・地面追従・特殊行動
src/input.js            … ポインタ/タッチ → セル配置（ブラシ）
src/render.js           … Canvas 描画（クレヨン質感・エフェクト）
src/ui.js               … 全消去ボタン
src/audio.js            … 効果音（任意）
src/main.js             … 起動とゲームループ
reference/design-options.html … アートの完成イメージ（案A参照用）
```

## 調整

ゲームの手ざわりは [`src/config.js`](./src/config.js) の `CFG` に集約しています
（セルサイズ・重力・車速・スポーン間隔・工場車両の割合など）。実機で触って調整してください。
